          Req) The globalized layout position is the display position in global space computed by adding the parent globalized layout position to the node’s local layout position.

        • Display Size (transient)
          Req) The display size is a dynamic attribute that maintains the display dimension of the node.
          Req) The display size shall be dynamically computed to include:
          • The node name and fold/unfold/iconize icons (header width);
          • The node ports and port names (left & right padding);
          • The visible children using their display position and size.
          Req) When unfold and with children, the display size of a node shall be calculated to contain the animated position and size of the children nodes.

        • Node Collisions

      • Anchor Relayout
        The anchor layout algorithm shall include the following elements:
        ◆ The anchor position of a drag node is set as-is;
        ◆ The anchor position of the children of a dragged node is relative to their parent anchor position;
        ◆ The anchor position of the parent is recomputed if the drag node causes the parent to be resized;
        ◆ Hierarchical grand-parent anchor positions are recomputed if a new anchor position/size of one of their children forces the grand-parent to be resized.
