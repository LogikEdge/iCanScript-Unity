      iCanScript is delivered in three Unity packages:
      - iCanScript: the core editor & engine (mandatory);
      - iCanScript_Nodes: base library nodes (optional);
      - iCanScript_Examples: example projects (optional).

  • Overview
    iCanScript is a visual programming editor that can be used standalone or to complement existing source code text editors.

    • iCanScript Anatomy
      iCanScript is composed of:
      * a central visual editor window;
      * a library tree panel;
      * a hierarchy tree panel;
      * an object instance wizard panel; and
      * a preference configuration panel.
      All of these windows and panels can be opened from Unity's Windows menu under the sub-menu iCanScript.
      Currently, the layout of iCanScript windows and panels are left to the user.

  • Saving & Loading Your Visual Scripts
    Your visual scripts are automatically saved and reloaded as part of the game object in which they reside. Your iCanScript components may be in one of two configurations: either as part of a GameObject in a Scene or as part of a Prefab in the Project.
